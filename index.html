读书笔记 |《逆转》

发表于 2017-03-13 | 分类于 读书笔记 |

关于本书：

《逆转》是一本关于如何面对巨人的书，这里的巨人指的是各种强大的对手，不幸和烦恼。每一章讲述的都是不同人的故事，他们面对巨大的挑战，并不得不做出回应。本书通过这些故事探究了两个观点：

  1. 那些在我们看来有价值的东西往往产生于不平衡的冲突当中。因为我们在希望渺茫时做出的反应往往包含让事情变得伟大而美丽的因素。
  2. 我们对于这些不平衡的冲突的理解往往是错误的。巨人并不是我们想象的那样，他们身上那些看似优势的东西常会变成劣势。事实上，身为弱者，却能在某些未知的方面改变人们。

关于作者：

马尔科姆·格拉德威尔（Malcolm Gladwell，1963年9月3日－）目前是《纽约客》杂志撰稿人及畅销作家。著作有：《引爆点》、《异类》、《大开眼界》、《眨眼之间》、《逆转》

核心内容：

一、优势中的劣势，劣势中的优势

  1. 事情本不像我们想的那样，我们要洞察本质，发现优势和劣势
  2. 倒U曲线：超过了某一个中间点，财富的增长并不会让我们的生活变得更加美好
  3. 大池中的小鱼，还是小池中的大鱼：小池中的大鱼可以避免大池中的条条框框，进而发挥自己的价值
  4. 相对剥夺：幸福国家的居民自杀率高于不幸福国家，幸福指数的高并不能说明其居民的幸福程度

二、值得经历的困难理论

  1. “不那么流畅”的问题反而能让人去思考更深层次的东西。他们会花上更多的精力，更深入的去挖掘，认真的去思考。当他们克服了这些障碍，他们就能做的更好。
  2. “资本学习”和“补偿学习”
  3. “局外人主义”会让你获得更多与“局内人”不同的能力，而这种能力正是你的优势所在。
  4. 惨痛的经历会给人们带来两种完全不同的影响。这种经历在深刻伤害某个群体的同时，也让另一个群体变得强大。
  5. 勇气不是与生俱来的，它是一种当你经历过痛苦才会得到的东西。
  6. “恶作剧精灵”：当你没有什么可以失去的时候，你会发现一片意料之外的自由之地。

三、权利的限制

  1. 合法性原则：

    1. 那些被要求服从权利机构的人必须要有发言的权利
    2. 法律要具备可预见性。
    3. 群里机构必须平等。不能其实任何一个群体。

  2. 当法律不具备合法性，人们便不会遵守法律，相反，他们会奋起反抗。

  3. 权利是有限的。使用权利时，必须让人们看到其合法性；使用不当的话，便会造成与预期效应相反的结果。
  4. 门诺派的哲学是“原谅”，然后继续往前走。因为他们相信，权利的效果是十分有限的。

摘抄：

  • 真正的力量，往往隐藏在柔弱的身躯中。力量和强大并不总是它们看起来的哪个样子。
  • 些人假装富有，其实他们一无所有；一些人假装贫穷，其实他们早已拥有巨大的财富。
  • 财富包含自我毁灭的因子。
  • 我的身上有一根刺，它是撒旦的信使，是来折磨我的。我求了上帝三次，求他拔掉那根刺。但是上帝对我说：“我的恩典足够你用的，我的力量会使弱点变得完美。”因此我要欣然夸耀我的弱点，这样基督的力量才会停留到我的身上。因为基督，我在弱点、羞辱、艰苦、迫害和困境中感到快乐。因为当我懦弱时，便是我强大的时候。

  • 发明家的三个特质：开放，尽责，不随和。

    • 开放：他们可以想出其他人想不到的东西，要愿意去挑战他们的先入为主之见。
    • 尽责：一个拥有好主意却缺乏将主意变成现实的自律和坚持的思想家只能是一个梦想家。
    • 不随和：不随和的人愿意承担社会风险，做那些别人可能不会认同的事情。

  • 我们所有人都不只会害怕，我们也会因为害怕而害怕，而克服恐惧会让我们产生狂喜的情绪。

  • 一个理智的人会让自己去适应社会，而不理智的人则会固执地让社会去适应自己。因此，这个社会的说有进步都依赖于这些不理智的人。
  • 在一个不公平的世界里，“骗子”不得已才当“骗子”，他们只不过是利用自己所得到的一切。—— 伯明翰运动。
  • 我又转念一想，见日光之下，快跑的未必能赢，力战的未必能胜，智慧的未必得粮食，明哲的未必得资财，灵巧的未必得喜悦。所临到众人的，是在乎当时的机会。
  • 如何实施惩罚和惩罚本身一样重要。
  • 扫荡一个小镇，消灭一群敌人，打击一场运动没有想象中那么容易。因为，权利没有其本身看起来那么强大，也没有特别弱小。

RSS是什么

RSS(Really Simple Syndication,简易信息聚合)是消息来源格式规范，用以聚合经常发布更新数据的网站，是一个让信息可以快速传播的平台。让每一个人成为潜在的信息传播者。发布一个RSS Feed，其中包含的信息可以被其他网站直接利用，由于这写数据包含的信息是XML格式的，也可以在手机等终端使用。
